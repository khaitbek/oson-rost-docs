## Rust o'zi nima?

Rust dasturlash tili high-level toollar orqali low-level kod yozish imkoniyatini beruvchi, 
juda ham tezkor va xotira bilan ishlashda dasturchilarga katta yordam  beruvchi dasturlash tili. Rust so'nggi bir necha yillarda **Javascript, Go, Python** kabi dasturlash tillarini ortda qoldirgan holda  [Stackoverflow](https://stackoverflow.com) dasturchilari tomonidan *eng sevimli dasturlash tili* deb topilgan va bu albatta bejizga emas.

## Nega aynan Rust dasturlash tili?

Rust dasturlash tili bizga quyidagi imkoniyatlarni berish orqali **xavfsiz**, **tez ishlaydigan**, **ishonchli** va **bir nechta tizimlarda ishlay oladigan** dasturlar yoza olishimizni ta'minlaydi:

- **xavfsizlik** - Istalgan dasturlash tilini o'rganish jarayonidagi eng og'riqli nuqtalardan biri bu xatoliklar bilan ishlash albatta. Har bir dasturlash tilining o'ziga yarasha xatoliklar bilan ishlash usullari bor, xuddi shunday Rustda ham. Ammo Rustda xatoliklar bilan ishlashda dasturchilarga juda ham qulayliklar yaratilgan, kodingizni ishga tushirib ulgurmasingiz matn muharririda xatoliklarni ko'ra olasiz. Bundan tashqari, Rust bir nechta dasturlash tillarida xatoliklar bilan ishlashda uchraydigan xatolarni o'rgangan va shu sababdan ham Rust bir nechta dasturlash tillarida uchraydigan kamchiliklarni yopa oladi. 
Misol uchun, Rust dasturlash tilida hozirgi zamonaviy dasturlash tillarida mavjud bo'lgan **Garbage Collector** yo'q. Yokida Rust dasturlash tilida xuddi **C** dasturlash tilidek dasturchi xotirani kerakli vaqtda tozalab turishi ham talab etilmaydi. Rust o'zining **ownership** nomli modeli orqali dasturchilarga xotira bilan ishlashda katta yordam beradi. 
- **tezlik** - Rust dasturlash tili **compile bo'ladigan** dasturlash tili hisoblanadi, ya'ni Rustda yozadigan ilovalarimiz to'g'ridan-to'g'ri mashina kodiga o'giriladi. Bundan tashqari, ta'kidlab o'tganimizdek, Rustda bir nechta dasturlash tillarida mavjud bo'lgan **Garbage Collector** ham yo'q. 
Rust dasturlash tili tezlik o'ta muhim bo'lgan joylarda ishlatiladi. Misol uchun, Rust allaqachon [Linux kernelida sinovdan o'tkazilishni boshlagan](https://www.youtube.com/watch?v=YyRVOGxRKLg) va [Windows kernelini C/C++ dan Rust dasturlash tiliga o'tkazish ishlari boshlangan](https://www.youtube.com/watch?v=YyRVOGxRKLg). 
Dunyodagi eng yaxshi operatsion tizimning va eng keng tarqalgan operatsion tizimning kernelining katta qismi qisqa vaqt ichida Rust orqali qayta yozib chiqilsa, ajab emas
- **ishonchlilik** - Windows va Linux asta-sekinlik bilan Rustni sinab ko'rishni boshlayotgani Rustning qanchalik ishonchli dasturlash tili ekanligini ko'rsatib beradi. Windows va Linux hazilakam narsalar emas, albatta. Rust esa aynan mana shunday o'ta katta xavfsizlik va ishonchlilik talab qilinadigan tizimlarga mos keladigan dasturlash tili hisoblanadi. 
- **bir nechta tizimlarda ishlay oladi** - Rustning eng yoqimli jihatlaridan biri shundaki, siz Rustni o'rganish orqali bir nechta tizimlarda ishlaydigan turli xil ilovalar yoza olasiz. Misol uchun, Rust orqali ushbu turdagi ilovalarni yozishingiz mumkin:
  - CLI (terminalda run bo'ladigan) ilovalar
  - OS (operatsion tizimlar)
  - Web Development
  - Game Development
  - AI & Machine Learning va hokazo
